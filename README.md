# EXPLAINABLE-AI-DRIVEN-DEEP-LEARNING-FRAMEWORK-FOR-INFRARED-BASED-SUBSTATION-EQUIPMENT-DIAGNOSIS
This project uses thermal images to identify high-voltage substation devices. It combines EfficientNet-B0 and LBP for feature extraction and uses XAI for transparency. The system improves fault detection, reduces maintenance cost, and enhances power system reliability.
